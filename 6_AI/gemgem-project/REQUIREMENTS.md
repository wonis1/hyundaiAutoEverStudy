Project: Game Hub "GemGem" Development Project
App name: GemGem

SDLC Phase 1 - Requirements (MVP)

Goal
- Map-based exploration app: move character, discover games, view details, launch in WebView.

Scope (MVP)
- Map exploration + joystick movement + camera follow (no zoom).
- Discovered game list panel only.
- Enter button -> detail modal -> Start -> WebView.
- Local JSON metadata.
- Login UI stub (auth=true).
- Themed locations per game.

Out of scope (MVP)
- Web game creation/hosting.
- Real OAuth/account management.
- Remote metadata/admin tools.
- Ads/payments, multiplayer.

Later
- Remote metadata.
- OAuth integration.
- Main page content blocks.
- Ranking/multiplayer expansion.

프로젝트: 게임 집합소 "겜겜" 개발 프로젝트
앱명: 겜겜

SDLC 1단계 - 요구사항 (MVP)

목표
- 맵 탐색형 앱: 캐릭터 이동 -> 게임 발견 -> 상세 확인 -> WebView 실행.

범위 (MVP)
- 맵 탐색 + 조이스틱 이동 + 카메라 추종(줌 없음).
- 발견한 게임만 목록 패널에 표시.
- 입장 버튼 -> 상세 모달 -> 시작하기 -> WebView.
- 로컬 JSON 메타데이터.
- 로그인 UI 스텁(auth=true).
- 게임별 테마형 장소.

범위 제외 (MVP)
- 웹게임 제작/호스팅.
- 실제 OAuth/계정 관리.
- 원격 메타데이터/관리자 도구.
- 광고/과금, 멀티플레이.

추후
- 원격 메타데이터.
- OAuth 연동.
- 메인 페이지 콘텐츠.
- 랭킹/멀티플레이 확장.
