# 장기(한글 장기) - 간단 기획/설계

## 목표
- React로 2인 로컬 장기 게임 구현.

## MVP 범위
- 9x10 보드 + 궁성 대각선.
- 말 이동 규칙, 턴, 잡기, 종료 판정.

## 핵심 규칙 요약
- 보드: 9x10.
- 궁성: 위/아래 3x3, 대각선 이동 가능.
- 말 이동:
  - 장군(G)/사(A): 궁성 내 1칸(대각 포함).
  - 상(E): 1칸 직진 + 2칸 대각, 중간 막힘 체크, 강 못 넘음.
  - 마(H): 1칸 직진 + 1칸 대각, 중간 막힘 체크.
  - 차(R): 직선, 궁성 대각 가능.
  - 포(C): 차처럼 이동, 잡을 때는 1개 뛰어넘기, 포는 포를 못 잡음.
  - 졸/병(S): 전진 1칸, 강 이후 좌우, 궁성 대각 가능.
- 턴: RED/BLUE 교대.

## 데이터 모델(초안)
- 좌표: { x:0..8, y:0..9 }
- 말: { id, type, side, pos }
- 보드: [10][9] 배열(말 id 또는 null)
- 상태: { pieces, board, turn, selected, history }

## 화면/구성
- 상단: 제목 + 현재 턴.
- 중앙: 보드.
- 우측(옵션): 수 기록/되돌리기.
